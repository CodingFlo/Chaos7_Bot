Befehle von Chaos7

Bot-Prefix: !
Befehl Kürzel Beschreibung
beleidigung beleidigen Ein Spaßbefehl mit dem man jemanden Beleidigen kann. (Es gibt mehrere Szenarien, die auftreten können als Beleidigung)
biddää - Ein Spaßbefehl mit dem man.... Joa gute Frage eigentlich... xDD
bug - Ein Spaßbefehl mit dem man mein Bot ein Statement zu seinen Bugs gibt.
cheater - Ein Spaßbefehl mit dem man jemanden als Cheater enttarnt und dem auf entsprechende Plattform canceln sollte.
command command Ein Befehl, der alle verfügbaren Befehle auflistet.
dc dc Ein Befehl mit dem man den Link zum Discord-Server von Flo bekommt.
dtest dtest Ein Spaßbefehl mit dem man testen kann, ob man schlauer oder dümmer als Flo ist.
dämlichkeitstest dätest Ein Spaßbefehl mit dem man testen kann, wie dähhmlich jemand ist. (bewusst mit H geschrieben)
fc - Ein Befehl mit dem man den Freundescode von Flo bekommt.
flotator - Ein Spaßbefehl mit dem ein Zeichen setzt, dass Flo seine Macht missbraucht.
followage - Ein Spaßbefehl mit dem ein überprüfen kann, wie lange jemand schon dem Kanal folgt, bzw. ob die Person überhaupt folgt.
geschenk - Ein Spaßbefehl mit dem ein jemanden ein Geschenk machen kann.
info - Ein Befehl, der einige Informationen bezüglich Chaos7 gibt.
itest - Ein Spaßbefehl der nur existiert, weil Meisti einen itest verlangte, bzw. weil das große I schwer von einem Kleinen L unterscheidbar ist.
jumpscare js Ein Soundalert Befehl mit dem man einen Jumpscare auslösen kann. (Wichtiger Hinweis: Der Jumpscare wird bewusst mit einer zufälligen Verzögerung ausgelöst, um den Überraschungseffekt zu erhöhen)
keks - Ein Spaßbefehl mit dem man zeigen kann, dass man mit jemanden seinen Keks teilen würde, wenn der Keks nur echt wäre
liebestest ltest Ein Spaßbefehl mit dem man testen kann, wie sehr jemand eine Person, bzw. sich selbst liebt.
lostheitstest lotest Ein Spaßbefehl mit dem man testen kann, wie lost jemand ist.
lurk - Ein Befehl mit dem man bescheid geben kann, dass man in den Lurk geht.
onlyfans of Ein Spaßbefehl für den Link zu meinem Discord-Server.
ranking rank Ein Befehl, der die fleißigsten Zuschauer der letzten 90 Tage anzeigt.
setgame game Ein Befehl rein für Mods, mit dem das gestreamte Game geändert werden kann
settitle title Ein Befehl rein für Mods, mit dem der Streamtitel geändert werden kann
shoutout so Ein Befehl rein für Mods um jemanden einen Shoutout zu geben
streik - Ein Spaßbefehl, der ein Zeichen setzt.
test - Ein Befehl mit dem man den testen kann, ob der Bot auf Befehle reagiert oder ein Fehler vorliegt.
timermessages tm Ein Befehl, der alle verfügbaren Timer-Nachrichten auflistet.
unlurk - Ein Befehl mit dem man mitteilen kann, dass man nun nicht mehr lurkt.
vod - Ein Befehl mit dem man den Link direkt zu Flos VOD Kanal bekommt.
ytb - Ein Befehl mit dem man den Link direkt zu Flos Best of Kanal bekommt.
ytlp - Ein Befehl mit dem man den Link direkt zu Flos LP Kanal bekommt.
votebann - Ein Spaßbefehl mit dem man zeigen kann, dass man jemanden gerne bannen würde.
voteja ja Wenn ein Voting aktiv ist, kann man damit abstimmen, dass man dafür ist.
votejain jain Wenn ein Voting aktiv ist, kanns man damit den Zufall entscheiden lassen, ob man dafür oder dagegen ist.
votenein nein Wenn ein Voting aktiv ist, kann man damit abstimmen, dass man dagegen ist.
votetimeout - Ein Spaßbefehl mit dem man zeigen kann, dass man jemanden gerne timeouten würde.
